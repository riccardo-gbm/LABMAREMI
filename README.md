# LABMAREMI

Website for "Labmaremi Cía. Ltda." — a B2B distributor of cleaning, disinfection, protection, and hygiene supplies based in Quito, Ecuador.

> Phase 1: professional frontend demo with mock data (no backend, no auth). Full spec in `docs/00-phase-1-scope.md`.

## Stack

React · Vite · TypeScript · Tailwind CSS v4 · shadcn/ui · React Router

## Setup

Requires Node.js 20+.

```bash
npm install
```

## Run

```bash
npm run dev
```

Open the URL Vite prints (default `http://localhost:5173`).

## Build

```bash
npm run build     # type-checks and builds to dist/
npm run preview   # serves the production build locally
```
